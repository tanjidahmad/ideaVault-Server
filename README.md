# IdeaVault Server

Backend API for the IdeaVault platform. This server provides secure authentication, idea management, comments, and user interaction features.
## Features

- Authentication & Security
  - Better Auth integration
  - JWT token verification
  - Protected API routes
  - Secure middleware implementation

- Database
  - MongoDB integration
  - Environment variable configuration

- Ideas Management
  - Create new ideas
  - Get all ideas
  - Update ideas
  - Delete ideas
  - Get idea details
  - Get user-specific ideas

- Search & Filtering
  - Search ideas by title
  - Filter ideas by category
  - Trending ideas section

- Comments & Interactions
  - Add comments
  - Update comments
  - Delete comments
  - View comments for specific ideas
  - User interaction history

- Server Features
  - Express.js backend
  - CORS support
  - REST API structure
